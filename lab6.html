Laboratorio: Análisis de Éxito de Productos

Archivos · Arreglos · Recursión · Formato tabular en Java

Objetivo

Leer un archivo con datos de productos, calcular un índice de éxito para cada uno, identificar los dos mejores, mostrar un informe tabular, y calcular el promedio general usando recursión.

Descarga del archivo de datos

Coloca productos.txt en el mismo directorio que tu programa (o cárgalo en JDoodle si usas STDIN).

Instrucciones (paso a paso)

  1. Crea tu proyecto Java (clase Main) en JDoodle o tu IDE.
  2. Lee el archivo:
    • Con archivo: Scanner in = new Scanner(new File("productos.txt"));
    • Con STDIN (JDoodle): Scanner in = new Scanner(System.in); y pega el contenido en el panel de stdin.
  3. Divide cada línea con split
    Explicación clave: split(String linea) divide una cadena de caracteres (o sea, un String) en segmentos usando un separador.
    Ejemplo: 
    String linea = "EcoBottle,78,2.5,85";
String[] partes = linea.split(","); // ["EcoBottle", "78", "2.5", "85"]
    Convierte partes[1], partes[2] y partes[3] a double.
  4. Guarda los datos en arreglos (sin colecciones): 
    String[] nombre = new String[MAX];
double[] interes = new double[MAX];
double[] costo = new double[MAX];
double[] positivos = new double[MAX];
double[] indice = new double[MAX];
int n = 0; // cantidad leída
  5. Calcula el índice de éxito por producto: 
    indice[i] = (0.5 * interes[i]) + (0.4 * positivos[i]) - (10 * costo[i]);
  6. Encuentra los dos mejores productos (top 1 y top 2) comparando valores en indice con bucles y variables max1/max2 (no uses sort ni colecciones).
  7. Imprime un informe tabular: 
    Producto        Interés  Positivos  Costo  Índice
--------------------------------------------------
...
--------------------------------------------------
1° lugar: <ProductoTop1>
2° lugar: <ProductoTop2>
    Sugerencia: usa System.out.printf para alinear columnas.
  8. Promedio general con recursión (sin loops en este método): 
    static double promedioRec(double[] a, int i, int n) {
    if (i == n - 1) return a[i];           // caso base
    double resto = promedioRec(a, i + 1, n); // recursión
    return (a[i] + resto * (n - i - 1)) / (n - i);
}
    En main: double promedio = promedioRec(indice, 0, n);
  9. Muestra el TOKEN de verificación (sirve como “huella” de tus cálculos): 
    int token = (int)(promedio + indice[max1] + indice[max2]);
System.out.println("TOKEN=" + token);
    Si cambian datos o hay errores de cálculo, el TOKEN también cambia.
  10. Entrega:
    • Envía la URL pública de JDoodle con tu programa que imprime la tabla, top 2, promedio y TOKEN.
    • Incluye al inicio del código: 
      // Nombre: ...
// TOKEN=...

Restricciones

  • Usa arreglos y bucles (for, while). Sin ArrayList, Streams ni ordenamientos automáticos.
  • Maneja líneas vacías con trim() y continue.
  • Formatea columnas con printf (ej: %-15s, %-8.2f).
